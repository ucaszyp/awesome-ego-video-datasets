# Egocentric video benchmarks & challenges

This file lists **evaluation benchmarks, diagnostic suites, and official challenge hubs** that are built on or centered on **egocentric / first-person video** (and closely paired ego–exo settings). It does **not** replace the dataset catalog: scale, licenses, and download details stay in [`datasets.md`](datasets.md). Match **Primary data** names to the summary table there.

**Ordering:** calendar **year (newest first)**; within each year, **name (A–Z)**.

**Notes column** uses short tags: *Standalone* = fixed benchmark task/splits; *Suite* = multi-track official challenge; *Dataset+benchmark* = corpus released together with primary eval tasks.
